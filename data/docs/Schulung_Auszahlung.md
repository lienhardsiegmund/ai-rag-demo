# Schulungsunterlage: Kreditauszahlung

Dieses Dokument dient ausschließlich der internen Schulung. Alle Daten sind fiktiv.

## Beispiel: Auszahlungsprozess
Nach der Bewilligung wird die Kreditsumme für die Antragstellerin Frau Anna Schneider
(Hauptstraße 45, 80331 München, IBAN DE12 3456 7890 1234 5678 90)
**in der Regel innerhalb von 2 Bankarbeitstagen** auf das angegebene Konto überwiesen.  

Falls eine Auszahlung auf ein externes Konto erfolgt, z. B. bei einer Überweisung zur Volksbank München,
kann sich die Auszahlung um zusätzliche 1–2 Werktage verzögern.

## Hinweise für Sachbearbeiter
- Prüfen Sie, ob alle Dokumente vollständig sind.
- Achten Sie darauf, dass sensible Daten **nicht an Dritte weitergegeben** werden.
- Bei Rückfragen wenden Sie sich bitte an die interne Kontaktstelle **Team Auszahlung**.
